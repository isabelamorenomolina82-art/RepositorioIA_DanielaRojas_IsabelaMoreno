# Comparación entre código Manual vs IA (GitHub Copilot)

## ✔️ Código generado por Copilot
- Más completo
- Refactorizado
- Documentado
- Maneja errores
- Más profesional

## ✔️ Código hecho manualmente
- Más simple
- Control total del estudiante
- Menos validaciones

## ✔️ Conclusión
Copilot produce código más robusto y profesional, mientras que el manual sirve como referencia sencilla.
