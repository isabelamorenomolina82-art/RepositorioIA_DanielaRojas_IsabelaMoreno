# Copilot 
GitHub Copilot es un asistente de codificación impulsado por inteligencia artificial (IA) que ofrece sugerencias de código en tiempo real, automatiza tareas repetitivas y ayuda a escribir, explicar, probar, documentar y desplegar código con mayor rapidez. Funciona como un "copiloto" de programación dentro del editor de código del desarrollador, permitiendo aumentar la productividad y enfocarse en resolver problemas complejos en lugar de realizar tareas rutinarias.

# Comparación entre Código Manual y Código Generado con IA (GitHub Copilot)

Este documento compara la calidad entre la versión de código escrita manualmente y la versión generada y refactorizada por GitHub Copilot.  

# Tabla Comparativa: Código Manual vs Código Generado con IA (GitHub Copilot)

| Criterio                    | Código Manual                                                                 | Código IA (Copilot)                                                                 |
|-----------------------------|-------------------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| **Eficiencia**              | Baja — usa algoritmo burbuja O(n²)                                           | Alta — usa `sorted()` con algoritmos optimizados                                    |
| **Líneas de código**        | Muchas líneas y bloques repetitivos                                          | Pocas líneas, solución limpia                                                       |
| **Legibilidad**             | Media — estructura menos clara                                               | Alta — código compacto y comprensible                                               |
| **Documentación**           | No incluye docstrings                                                        | Copilot genera documentación automática                                              |
| **Mantenibilidad**          | Difícil de extender o modificar                                              | Fácil de mantener gracias a buenas prácticas                                         |
| **Probabilidad de errores** | Alta — manipulación manual de índices y swaps                                | Baja — se delega a funciones internas probadas                                       |
| **Buenas prácticas**        | Limitadas                                                                    | Sigue estándares profesionales (type hints, docstrings, funciones integradas)        |
| **Escalabilidad**           | Baja                                                                          | Alta                                                                                  |
| **Robustez**                | Menor robustez, posible comportamiento inesperado                            | Muy robusto — usa funciones del lenguaje optimizadas y probadas                     |
| **Tiempo de desarrollo**    | Más lento — requiere escribir toda la lógica a mano                          | Más rápido — Copilot genera y mejora código automáticamente                          |
